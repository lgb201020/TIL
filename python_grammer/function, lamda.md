# 함수
- 함수란 특정 작업을 하나의 단위로 묶어 놓은 것을 의미한다.
- 함수를 사용하면 불필요한 소스코드의 반복을 줄일 수 있다.
- 내장 함수: 파이썬에서 기본적으로 제공하는 함수
- 사용자 정의 함수: 개발자가 직접 정의하여 사용할 수 있는 함수
## 함수 정의
- 프로그램에는 똑같은 코드가 반복적으로 사용되는 경우가 많다. 따라서 함수를 사용하면 소스코드의 길이를 줄일 수 있다.
    ```
    def 함수명(매개변수):
        실행 코드
        return 반환 값
    ```
  - 매개변수: 함수 내부에서 사용할 변수
  - 반환 값: 함수에서 처리 된 결과를 반환

- 예제
    ```
    def add(a,b):
        print('함수의 결과:', a+b)

    add(3,7)
    add(b = 3, a = 7) 
    # 파라미터의 변수를 직접 지정할 수 있으며 이 경우 매개변수의 순서가 달라도 상관없다.
    ```

## global 키워드
- global 키워드로 변수를 지정하면 해당 함수에서는 지역 변수를 만들지 않고, 함수 바깥에 선언된 변수를 참조하게 된다.
    ```
    a = 0

    def func():
        global a
        a += 1

    for i in range(10):
        func()

    print(a)
    ```
    ```
    10
    ```

- 값을 변경하거나 새로운 값을 대입하는 것이 아니라 단순히 전역변수로 선언된 값을 단순히 출력하는 등의 참조하는 경우 global 키워드 없이도 수행된다.
    ```
    a = 0

    def func():
        print(a+20)

    func()
    ```
    ```
    20
    ```
- 전역변수로 선언된 리스트의 경우 함수 내부에서 해당 리스트를 호출하는 경우 문제 없이 참조가 가능하며 값 수정이나 추가, 삭제 등이 가능하다.
    ```
    a = [1,2,3,4,5,6]

    def func():
        a.append(20)
        print(a)

    func()
    ```
    ```
    [1,2,3,4,5,6,20]
    ```

- 단 함수 안에서 전역변수와 지역변수의 이름이 동일하다면 지역변수를 우선적으로 참조한다.
    ```
    a = [1,2,3,4,5,6]

    def func():
        a = [1,2]
        a.append(20)
        print(a)

    func()
    print(a)
    ```
    ```
    [1,2,20]
    [1,2,3,4,5,6]
    ```
## 여러 개의 반환 값
- 파이썬에서 함수는 여러 개의 반환 값을 가질 수 있고 이 경우 다른 언어와 달리 여러개의 반환 값을 한 번에 return 가능하다.
    ```
    def operator(a,b):
        add_var = a + b
        subtract_var = a - b
        multiply_var = a * b
        divide_var = a / b
        return add_var, subtract_var, multiply_var, divide_var

    a, b, c, d = operator(7,3)
    print(a,b,c,d)
    ```
    ```
    10 4 21 2.333333333335
    ```
## 람다 표현식
- 람다 표현식을 이용하면 함수를 간단하게 작성할 수 있다. 
  - 특정한 기능을 수행하는 함수를 한 줄에 작성할 수 있다는 점이 특징이다.

    ```
    def add(a,b):
        return a+b

    # 일반적인 add() 메서드 사용
    print(add(3,7))

    # 람다 표현식으로 구현한 add() 메서드
    print((lamda a,b: a+b)(3,7))
    ```
    ```
    10
    10
    ```
- 람다 표현식은 어떠한 함수의 반환값을 입력으로 받는 또 다른 함수가 있는 경우 유용하게 사용되며 또한 함수가 매우 간단하거나 1회성으로 사용할 경우 람다 표현식을 쓰면 소스코드의 길이를 줄일 수 있다.
    ```
    array = [('홍길동', 50), ('이순신', 32), ('아무개', 74)]

    def my_key(x):
        return x[1]

    print(sorted(array, key=my_key))
    print(sorted(array, key=lamda x:x[1]))
    ```
    ```
    [('이순신', 32), ('홍길동', 50), ('아무개', 74)]
    [('이순신', 32), ('홍길동', 50), ('아무개', 74)]
    ```
- 여러개의 리스트에 동일한 규칙의 함수를 적용할 때 람다 표현식을 활용한다.
    ```
    list1 = [1,2,3,4,5]
    list2 = [6,7,8,9,10]

    result = map(lamda a,b: a + b, list1, list2)

    print(list(result))
    ```
    ```
    [7,9,11,13,15]
    ```
