# 조건문
- 조건문은 프로그램 흐름을 제어하기 위한 문법으로 조건에 따라 프로그램 로직을 설정할 수 있다.
- 조건문은 아래와 같이 작성할 수 있다.
    ```
    if 조건식1:
        실행코드
    ```
    ```
    if 조건식1:
        실행코드
    else:
        실행코드
    ```
    ```
    if 조건식1:
        실행코드
    elif 조건식2:
        실행코드
    else:
        실행코드
    ```
- 예제: 성적 구간에 따른 학점 출력
  ```
  score = 85

  if score >= 90:
      print("학점: A")
  elif score >= 80:
      print("학점: B")
  elif score >= 70:
      print("학점: C")
  else:
      print("학점: F")
  ```
  ```
  학점: B
  ```

## 비교 연산자
- 비교 연산자는 특정한 두 값을 비교할 때 이용할 수 있다.
    |비교 연산자| 설명|
    |------|---|
    |X == Y|X와 Y가 서로 같을 때 참(True)|
    |X != Y|X와 Y가 서로 다를 때 참(True)|
    |X > Y|X가 Y보다 클 때 참(True)|
    |X < Y|X가 Y보다 작을 때 참(True)|
    |X >= Y|X가 Y보다 크거나 같을 때 참(True)|
    |X <>= Y|X가 Y보다 작거나 같을 때 참(True)|
  - 대입 연산자(=)와 같음 연산자(==)의 차이점에 유의해야한다.

## 논리 연산자
- 논리 연산자는 논리 값(True/False) 사이의 연산을 수행할 때 사용된다. 이때 C++이나 JAVA와 달리 python은 &, |과 같은 기호가 아닌 단어를 직접 연산자로 사용한다.
    |논리 연산자| 설명|
    |------|---|
    |X and Y|X와 Y가 모두 참(True)일 때 참(True)|
    |X or Y|X와 Y 중에 하나만 참(True)이어도 참(True)|
    |not X|X가 거짓(False)일 때 참(True)|

## 파이썬의 기타 연산자
- 다수의 데이터를 담는 자료형을 위해 in 연산자와 not in 연산자가 제공된다.
  |연산자| 설명|
    |------|---|
    |x in 리스트|리스트 안에 x가 들어있을 때 참(True)|
    |x not in 리스트|리스트 안에 x가 들어있지 않을 때 참(True)|
  - in, not in 모두 리스트, 튜플, 문자열, 딕셔너리 모두에서 사용 가능하다.

## 파이썬의 pass 키워드
- 아무것도 처리하고 싶지 않을 때 pass 키워드를 사용한다.
- 대표적으로 디버깅 과정에서 많이 사용된다.

## 조건문의 간소화
- 조건문에서 실행될 소스코드가 한 줄인 경우, 굳이 줄 바꿈을 하지 않고도 간략하게 조건문을 쓸 수 있다.
    ```
    score = 85

    if score >= 80: result = "PASS"
    else: result = "FAIL"
    ```
- 조건부 표현식(conditional expression)은 if ~ else 문을 한 줄에 작성 할 수 있도록 해준다. 단 if가 중간에 들어간다.
    ```
    score = 85
    result = "PASS" if score >= 80 else "FAIL"
    ```
- 파이썬에서는 다른 프로그래밍 언어와 다르게 조건문 안에서 수학의 부등식을 그대로 사용할 수 있다.
  - 파이썬에서는 x>0 and x<20 과 0 < x < 20은 모두 같은 의미이다.
